# Text Analysis Tool

A Java console application designed to perform statistical analysis on text input.

This tool allows users to input paragraphs or lengthy text documents and immediately generates a comprehensive report. 
## Demo

https://github.com/user-attachments/assets/f30fe74c-c609-48e9-a4dd-584e88b564bc

## Features

* **Input Processing:** Handles large blocks of text input from the console.
* **Data Summary:** Displays total words and characters instantly.
* **Frequency Analysis:**
    * Automatically finds the most common character.
    * Calculates the frequency of any specific character requested by the user.
    * Calculates the frequency of any specific word requested by the user.## Installation

Ensure you have the Java JDK installed (Version 22 or higher recommended).

1. Clone the repository:
```bash
git clone [https://github.com/yourusername/text-analysis-tool.git](https://github.com/yourusername/text-analysis-tool.git)
```

2. Navigate to the project directory and compile the code:

```bash
javac Main.java
```

3. Run the application:

```bash
java Main
